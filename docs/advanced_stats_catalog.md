# 심화통계 마스터 카탈로그 (작성 중)

> 통계보기 고도화용 내부 문서. 외부 강의 자료의 조회 세팅을 그대로 재현하는 것이 목표.
> 데이터 소스는 전부 공공 통계(KOSIS·국토교통부·한국부동산원)이며, 사이트에는 출처 기관명만 표기한다.

## 표기 규칙

- **지역 그룹(공통)**: 전국 / 수도권(서울·인천·경기) / 지방소계 / 기타광역시(부산·대구·광주·대전·울산) / 기타지방 — 시도 개별값 병기
- 갱신 주기: KOSIS 자료갱신일 기준

- **40제외 인허가** = A4의 `계 − 40㎡이하` (초소형 오피스텔·도시형 제외 목적, 실질 아파트 공급 근사)
- **반기 물량** = 6월 누계(상반기), 12월 누계 − 6월 누계(하반기)
- 지역별 **과거 저점·고점 참조선**: 수도권 206,446(2014)~266,034(2007) / 부산 15,160~30,275 / 대구 16,512~26,121 / 광주 5,261~13,309 / 대전 4,704~14,024 / 울산 5,822~11,722 / 세종 2,917~12,984 / 강원 11,355~17,525 / 충북 10,472~26,416 / 충남 22,095~31,540 / 전북 11,075~17,473 / 전남 14,101~18,605 / 경북 10,403~26,763 / 경남 21,791~36,382 / 제주 4,306~9,890 (연간, 40제외 기준)
- 해석 축: 인허가→입주 4~6년 시차. 저점 미만 반기가 이어지면 몇 년 뒤 공급절벽.

**HWP 표 재현 대상**: 반기별(01-06/07-12/연간) × 시도별 매트릭스, 2007~현재.

## B. 월간 가격동향 (확정 — HWP '2026 04월' + 2019 영상 화면 검증)

**원본**: KOSIS > 한국부동산원 「전국주택가격동향조사」 (아파트 기준)

| # | 지표 | 사용 세팅 |
|---|---|---|
| B1 | 매매가격지수 | 아파트, 시도별, 월간 — 지수 + 전월대비 변동 |
| B2 | 전세가격지수 | 〃 (장기 시계열 2005~ 도 사용, 기준시점 환산 주의) |
| B3 | 월세가격지수(월세통합) | 〃 |
| B4 | 매매가격 대비 전세가격 비율(전세가율) | 아파트, 시도별 — **KOSIS orgId=408, tblId=DT_30404_N0006_R1 계열**('유형별 매매가격 대비 전세가격 비율', 2019 화면에서 URL 직접 확인). 즐겨찾기로 두고 반복 확인하는 핵심 표 |
| B5 | 전월세전환율 | 아파트, 시도별 |

**핵심 해석 축(2019 시리즈)**: 전세가율은 '절대 수치'보다 **방향**이 먼저다(전세가격 상승 국면이라는 전제 위에서만 유효). 수도권은 60% 후반대, 지방은 70~85%대에서 매매 전환(턴) — 상승 기대감이 있는 지역은 더 낮은 전세가율에서 턴. **실거주비용 = 전세가율 × (대출금리 vs 전월세전환율 비교)** — 전월세전환율이 대출금리보다 높으면 자가보유가 유리해지는 지점.

**월간 리포트 레이아웃(HWP 재현)**: 권역 3블록(수도권 4열 / 광역시 6열 / 도 8열) × 5행(월세/전세/매가/전세가율/전환율), 각 셀 = 지수값 + 월변동.
**해석 축**: 전세>월세>매매 순 상승 = 공급부족 에너지가 전세에 집중 → 전세가율 상승 → 시차 두고 매매 전이. 금리가 상승 억제 변수.

## C. 입주물량 (확정 — HWP '호갱노노 입주멸량')

**원본**: 호갱노노 입주물량 (공공 API 아님 — 수동 집계). 분기별 × 시도별, 2017~2029(미래 포함).

- 지역별 **수요 기준선**: 수도권 50,000→55,000 / 부산 6,000→7,000 / 울산 2,000→2,500 / 경남 6,000 / 대구 5,000 / 경북 5,000 / 광주 2,700→3,000 / 전남 3,000 / 대전 2,200→3,000 / 충남 3,500→4,000 / 충북 2,700→2,400 / 전북 2,500 / 강원 2,600→3,000 (분기당, 시기에 따라 상향)
- 특이 대단지 주석: 김해, 배방, 음성, 오송, 우암, 대연, 도안, 탕정, 성성, 청주 등
- 참고: 우리 사이트는 KOSIS 주택준공실적으로 근사 가능(과거분). 미래분은 별도 소스 필요.

## D. 주간 동향 (부분 확정 — 2024 영상 1편 47~49분)

**원본**: 한국부동산원 R-ONE(부동산통계뷰어) > 전국주택가격동향조사 > **주간아파트동향 > 매매가격지수** (+ 전세)
- 용도: 월간보다 빠른 속보 확인. "급박할 때 주간을 본다."
- 음성파일 제목("주간 가격 인허가를 보는 이유")과 직결 — 전사 완료 후 세부 보강.

## 구현 방침 (통계보기 메뉴 고도화)

1. 통계 뷰를 **기본통계 / 심화통계** 2개 탭(또는 세그먼트)으로 분리. 기존 표는 전부 기본통계로.
2. 심화통계 1차 구현(정적 데이터 → 추후 KOSIS OpenAPI):
   - 인허가(반기별·40제외·저점/고점 참조선) — HWP 데이터로 시딩
   - 월간 가격동향 5종 세트(매매·전세·월세지수, 전세가율, 전환율) — HWP '2026 04월' 형식
   - 입주물량(분기별, 수요 기준선) — 호갱노노 집계 시딩
3. 반응형: **모바일 = 요약 카드**(지역 그룹 접기, 핵심 수치만) / **PC = 상세 매트릭스**(HWP 표 그대로).
4. KOSIS OpenAPI 연동 시 표 ID: 인허가 규모별(주택건설실적통계), 전세가율 DT_30404_N0006_R1 계열부터.
